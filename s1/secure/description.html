Cyber Shield

Welcome!

Cyber Shield is a model used to detect phishing emails and report to the user about any suspicious emails. Our Cyber Shield runs continuously whenever the PC is powered on. Without any manual intervention, it gets auto-started.

Points to follow:

  • You have to upload the extension by going to Google Extensions. Then, load the extension.
  • Once the extension is loaded, click on the extension icon. You will have 3 options:
    • Review: Used to check the phishing mails detected.
    • Details: Provides information about the extension's permissions.
    • Description: Provides information on using Cyber Shield.
  • When email phishing detection starts, it will ask for an email and password. Enter these and click submit. Phishing mails are detected every 10 seconds, and a report is generated for detected phishing mails. Users can download and view these reports.

How to create an app password for Gmail:

  • Go to your Google Account and select Security.
  • Under Signing in to Google, select App passwords. You may need to sign in again.
  • At the bottom, choose Select app and choose the app you’re using.
  • Choose Select device and choose the device you’re using.
  • Select Generate.
  • Follow the instructions to enter the app password. The 16-character code will appear. Copy and use this password in your application.

Note:

Internet Connection is compulsory for the Cyber Shield extension to function properly.
